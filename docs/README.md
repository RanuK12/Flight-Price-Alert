# Screenshots

Place the following screenshots in this folder to display them in the README:

| Filename | Description | Suggested size |
|----------|-------------|----------------|
| `dashboard-preview.png` | Full dashboard overview (header + search + deals) | 1440×900 |
| `dashboard-search.png` | Search form and results section | 1280×720 |
| `telegram-alert.png` | Telegram notification message example | 400×600 |

## How to capture

1. **Dashboard**: Open `http://localhost:4000`, take a full-page screenshot.
2. **Search**: Perform a search on the dashboard, capture the result card.
3. **Telegram**: Forward a deal notification from the bot in Telegram, screenshot the chat.
